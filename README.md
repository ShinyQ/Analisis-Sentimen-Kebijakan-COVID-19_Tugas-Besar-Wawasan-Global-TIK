# Analisis-Sentimen-Kebijakan-COVID-19_Tugas-Besar-Wawasan-Global-TIK
Analisis sentimen dari masyarakat Indonesia terhadap kebijakan pemerintah mengenai vaksin COVID-19 yang akan dibagikan secara gratis kepada masyarakat Indonesia.  Tujuan ini berguna bagi Pemerintah Indonesia untuk mengetahui kepercayaan masyarakat terhadap vaksin COVID-19 yang akan mulai dibagikan.
